# vscode-base64

Forked from https://github.com/adamhartford/vscode-base64

Quickly base64 encode/decode your text selections.

## Commands:

- **Base64 Encode**
- **Base64 Decode**

## Keybindings:

### Mac

- **cmd+e cmd+e** Encode
- **cmd+e cmd+d** Decode

### Win/Linux

-  **ctrl+e ctrl+e** Encode
-  **ctrl+e ctrl+d** Decode
